## 版本 2.5.0 (2026-05-27)

### ✨ 新增功能

#### 用户文档访问记录查询
- 📋 **查询最近浏览/编辑的文档**：支持查询当前用户最近浏览或编辑的文档列表
  - `--action recent-view`：查询最近浏览的文档
  - `--action recent-edit`：查询最近编辑的文档
  - 支持时间范围筛选和分页查询
  - 最多返回 200 条记录，避免系统负载过高

## 版本 2.2.0 (2026-05-07)

### ✨ 新增功能

#### 数据表管理能力完整集成
- 📊 **数据表字段管理**：支持获取、添加、删除数据表字段
- 📝 **数据表记录管理**：支持获取、添加、更新、删除数据表记录（支持批量操作）
- 👁️ **数据表视图管理**：支持获取数据表所有视图信息
- ➕ **数据表创建**：支持在文档中创建新数据表

### 📋 核心能力清单

#### 文档管理类（5 项）
| 功能 | 命令 |
|-----|------|
| 查询文档内容 | `query-content` |
| 查询知识库文档列表 | `query-repo` |
| 创建文档（支持图片上传） | `create-doc` |
| 编辑文档正文 | `edit-content` |
| 复制文档 | `copy-doc` |

#### 权限管理类（4 项）
| 功能 | 命令 |
|-----|------|
| 查询用户对文档的权限 | `query-permission` |
| 添加文档成员 | `add-member` |
| 更新成员角色 | `update-member` |
| 修改文档公开范围 | `change-scope` |

#### 评论互动类（2 项）
| 功能 | 命令 |
|-----|------|
| 查询文档评论（含批注） | `query-comments` |
| 创建文档评论/批注 | `create-comment` |

#### 互动数据 & 用户信息类（4 项）
| 功能 | 命令 |
|-----|------|
| 查询文档浏览记录 | `query-recent-view` |
| 导出流程图数据 | `query-flowchart` |
| 查询用户信息（含个人知识库ID） | `query-user-info` |
| 上传附件 | `upload-attachment` |

#### 表格导入导出类（2 项）
| 功能 | 命令 |
|-----|------|
| 导出在线表格为 Excel | `export-sheet` |
| 导入 Excel 为在线表格 | `import-sheet` |

#### 数据表管理类（9 项）
| 功能 | 命令 |
|-----|------|
| 获取数据表字段列表 | `get-datasheet-fields` |
| 添加数据表字段 | `add-datasheet-field` |
| 删除数据表字段 | `del-datasheet-field` |
| 获取数据表记录（支持分页/筛选/排序） | `get-datasheet-records` |
| 添加数据表记录（支持批量） | `add-datasheet-records` |
| 更新数据表记录（支持批量） | `update-datasheet-records` |
| 删除数据表记录（支持批量） | `delete-datasheet-records` |
| 获取数据表视图列表 | `get-datasheet-views` |
| 创建新数据表 | `create-datasheet` |

### 🛠️ 使用方式

```bash
# 环境变量要求
export SANDBOX_USERNAME="your_username"

# 执行命令
$SKILL_DIR/bin/ku <subcommand> [options]
```
